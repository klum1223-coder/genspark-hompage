```bash
# 히스토리에서 제거
git filter-branch --force --index-filter \
  "git rm --cached --ignore-unmatch .env" \
  --prune-empty --tag-name-filter cat -- --all

# 강제 푸시
git push origin main --force
```

### Q2. 푸시가 거부당했어요
```bash
# 원격 변경사항 가져오기
git pull origin main --rebase

# 푸시
git push origin main
```

### Q3. 커밋을 취소하고 싶어요
```bash
# 마지막 커밋 취소 (파일 유지)
git reset --soft HEAD~1

# 마지막 커밋 완전 삭제
```
