### 2. ✅ 관리자 페이지 - 담임목사 탭 개선

#### 2-1. PastorInfo 인터페이스 업데이트
```typescript
interface PastorInfo {
  name: string
  education: string[]
  career: string[]        // ✅ 추가
  message: string
  photo?: string          // ✅ 추가
}
```

#### 2-2. 경력 입력 필드 추가
**관리자 페이지** → **👨‍🏫 담임목사** 탭

**새 필드**:
```
경력 (줄바꿈으로 구분)
┌─────────────────────────────┐
│ 희망도서관 청주지부장         │
│ 글쓰기 운동본부 충북지역장    │
└─────────────────────────────┘
예: 희망도서관 청주지부장
```

**사용 방법**:
1. 각 경력을 한 줄씩 입력
2. Enter로 줄바꿈
3. 저장 버튼 클릭
4. 교회소개 페이지에 자동 반영

### 3. ✅ 관리자 페이지 - 교회사역 이미지 등록 기능

#### 3-1. Ministry 인터페이스 업데이트
```typescript
interface Ministry {
  id: string
  title: string
  icon: string
  description: string
  detailContent: string
  image?: string          // ✅ 추가
}
```

#### 3-2. 이미지 URL 입력 필드
**관리자 페이지** → **🎯 교회사역** 탭

**새 필드**:
```
사역 이미지 URL (선택사항)
┌─────────────────────────────────────────┐
│ https://example.com/image.jpg            │
└─────────────────────────────────────────┘

[이미지 미리보기]
```

**기능**:
- ✅ 이미지 URL 입력
- ✅ 실시간 미리보기 (32x32 썸네일)
- ✅ 이미지 로드 실패 시 자동 숨김
- ✅ 상세 내용에 블로그 URL도 입력 가능

#### 3-3. 이미지 표시
**메인 페이지**:
- 이미지 URL이 있으면 카드에 이미지 표시
- 없으면 아이콘만 표시

**교회사역 페이지**:
- 상세 페이지에서 큰 이미지 표시 가능
- 블로그 링크와 함께 사용 가능

### 4. ✅ 온라인 등록 페이지 검증

#### 4-1. 현재 상태
**정상 작동 확인 완료** ✅

**주요 기능**:
- ✅ 등록 유형 선택 (새가족, 행사, 기도, 상담)
- ✅ 필수 필드 검증 (name, phone, email)
- ✅ 개인정보 동의 체크박스 필수
- ✅ 제출 중 버튼 비활성화
- ✅ 성공/실패 메시지 표시

#### 4-2. 필수 필드 검증
```tsx
{/* 이름 */}
<input 
  type="text"
  name="name"
  required           // ✅ HTML5 필수 검증
  placeholder="홍길동"
/>

{/* 연락처 */}
<input 
  type="tel"
  name="phone"
  required           // ✅ HTML5 필수 검증
  placeholder="010-1234-5678"
/>

{/* 이메일 */}
<input 
  type="email"
  name="email"
  required           // ✅ HTML5 필수 검증
  placeholder="example@email.com"
/>

{/* 개인정보 동의 */}
<input 
  type="checkbox"
  name="agreement"
  checked={formData.agreement}
  onChange={handleChange}
/>
```

#### 4-3. 제출 버튼 상태
```tsx
<button
  type="submit"
  disabled={isSubmitting}    // 제출 중에만 비활성화
  className="..."
>
  {isSubmitting ? '등록 중...' : '등록하기'}
</button>
```

**동작**:
1. 사용자가 필수 필드 입력
2. 개인정보 동의 체크
3. "등록하기" 버튼 활성화
4. 클릭 시 "등록 중..." 표시 및 버튼 비활성화
5. 완료 후 성공 메시지 표시

#### 4-4. 체크 포인트
- ✅ 필수 필드 미입력 시 브라우저 기본 검증 메시지
- ✅ 개인정보 동의 미체크 시 커스텀 경고 메시지
- ✅ 제출 중 중복 클릭 방지
- ✅ 제출 완료 후 폼 초기화

5. **개인정보 수집 및 이용 동의** 체크
