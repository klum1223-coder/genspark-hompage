## 11. 메인 배너 관리

### 11.1 이동 경로
관리자 페이지 → **🎬 메인배너** 탭

### 11.2 텍스트 수정
- **메인 제목**: 하나님의 사랑으로\n함께하는 공동체
  - `\n`을 입력하면 줄바꿈됩니다
- **부제목**: 예수 그리스도의 복음으로 세워진
- **설명**: 생명과 소망이 넘치는 교회

### 11.3 배경 설정

#### 그라데이션 (기본)
1. **배경 타입**: 그라데이션 선택
2. 기본 갈색 계열 그라데이션 적용
3. **💾 저장하기** 버튼 클릭

#### 이미지 배경
1. **배경 타입**: 이미지 선택
2. **배경 이미지 URL** 입력:
   ```
   https://images.unsplash.com/photo-1438032005730-c779502df39b?w=1920&q=80
   ```
3. 추천 이미지 크기: 1920x1080px 이상
4. **💾 저장하기** 버튼 클릭

#### 비디오 배경
1. **배경 타입**: 비디오 선택
2. **배경 비디오 URL** 입력 (MP4 형식):
   ```
   https://example.com/church-video.mp4
   ```
3. 비디오는 자동 재생/반복/무음으로 설정됩니다
4. **💾 저장하기** 버튼 클릭

### 11.4 무료 이미지 사이트
- **Unsplash**: https://unsplash.com
- **Pexels**: https://www.pexels.com
- **Pixabay**: https://pixabay.com

## 13. 자주 묻는 질문 (FAQ)

### Q1. 저장한 내용이 메인 페이지에 반영되지 않아요
**A**: 메인 페이지를 **새로고침**(F5)해주세요. 브라우저 캐시 때문에 이전 내용이 표시될 수 있습니다.

### Q2. 비밀번호를 잊어버렸어요
**A**: 비밀번호는 `joosung2025`입니다. 변경이 필요하면 개발자에게 문의하세요.

### Q3. 이미지를 어떻게 업로드하나요?
**A**: 현재 버전은 이미지 URL을 입력하는 방식입니다:
1. 이미지를 무료 호스팅 사이트에 업로드 (Imgur, Cloudinary 등)
2. 이미지 URL을 복사하여 관리자 페이지에 붙여넣기

### Q4. 유튜브 영상이 표시되지 않아요
**A**: 
- 유튜브 URL이 올바른지 확인하세요
- 영상이 "공개" 또는 "일부 공개"로 설정되어 있는지 확인하세요
- "비공개" 영상은 임베드할 수 없습니다

### Q5. 모바일에서 페이지가 이상하게 보여요
**A**: 
- 브라우저 캐시를 삭제해보세요
- 다른 브라우저에서 테스트해보세요
- 문제가 지속되면 스크린샷과 함께 개발자에게 문의하세요

### Q6. 데이터가 갑자기 사라졌어요
**A**: 
- 브라우저의 "방문 기록 삭제"에서 "사이트 데이터" 삭제 시 데이터가 사라질 수 있습니다
- **예방**: 정기적으로 데이터를 백업하세요
  1. F12 → Application → Local Storage
  2. 각 key의 value를 복사하여 텍스트 파일로 저장

### Q7. 관리자 페이지에 접속할 수 없어요
**A**:
- URL이 올바른지 확인: `/admin`
- 브라우저 쿠키 설정 확인
- 다른 브라우저나 시크릿 모드로 시도

### Q8. 스타일 변경이 전체 페이지에 적용되나요?
**A**: 네, 스타일 설정은 홈페이지 전체에 적용됩니다.

### Q9. 여러 사람이 동시에 관리자 페이지를 사용할 수 있나요?
**A**: 가능하지만, 동시에 같은 항목을 수정하면 나중에 저장한 내용이 최종 반영됩니다.

### Q10. 삭제한 내용을 복구할 수 있나요?
**A**: 현재 버전은 복구 기능이 없습니다. 삭제 전에 신중하게 확인하세요.

   - `hero_content`
