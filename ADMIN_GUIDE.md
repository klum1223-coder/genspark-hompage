## 📋 관리 가능한 콘텐츠

### 1. 🏛️ 교회 기본 정보
**수정 가능 항목:**
- 교회명 (한글/영문)
- 대표 전화
- 팩스
- 이메일
- 주소 (기본 주소 + 상세 주소)

**저장 방법:**
1. 각 항목 입력 후
2. 하단의 **"저장"** 버튼 클릭
3. 초록색 메시지 확인: "✓ 저장되었습니다!"

**반영 확인:**
- Footer (하단)
- Contact 페이지 (오시는 길)
- About 페이지 (교회 소개)

---

### 2. 👨‍🏫 담임목사 정보
**수정 가능 항목:**
- 담임목사 성함
- 학력 (여러 줄 입력 가능)
- 목회 철학 / 메시지

**입력 팁:**
```
학력 입력 예시:
호서대학교 신학과 졸업 (B.A.)
서울신학대학교 대학원 목회학 석사 (M.Div.)
```

**반영 확인:**
- About 페이지 > 담임목사 소개 섹션

---

### 3. ⛪ 예배 시간
**수정 가능 항목:**
- **주일 예배**: 1부, 2부, 찬양예배
- **주중 예배**: 수요예배, 새벽기도회, 금요기도회

**각 예배별 설정:**
- 예배 이름 (예: 1부 예배)
- 시간 (예: 오전 09:00)
- 설명 (예: 장년 예배)

**반영 확인:**
- 메인 페이지 > 예배 시간 섹션
- Footer (하단) > 예배 시간
- Contact 페이지 > 예배 시간

---

### 4. 📢 팝업 배너 관리
**완전한 제어 가능!**

**설정 항목:**
- ✅ **활성화/비활성화**: 체크박스로 켜기/끄기
- 팝업 제목
- 팝업 내용 (여러 줄 가능)
- 버튼 텍스트
- 버튼 링크

**사용 예시:**
```
제목: 2024 신년 부흥회
내용:
새해를 맞이하여 은혜로운 부흥회를 개최합니다.

📅 일시: 2024년 1월 15일(월) ~ 17일(수)
⏰ 시간: 매일 저녁 7시 30분
📍 장소: 본 교회 대예배실

버튼 텍스트: 자세히 보기
버튼 링크: /news
```

**팁:**
- 체크박스를 해제하면 팝업이 표시되지 않습니다
- 미리보기로 저장 전에 확인하세요
- 사용자가 "오늘 하루 보지 않기"를 선택하면 자정까지 표시 안 됨

### 5. 🎨 메인 배너 (히어로 섹션)
**홈페이지 첫 화면의 큰 배너를 수정할 수 있습니다!**

**수정 가능 항목:**
- 메인 제목 (여러 줄 가능)
- 부제목 (상단)
- 설명 (하단)

**입력 예시:**
```
메인 제목:
하나님의 사랑으로\n함께하는 공동체

부제목:
예수 그리스도의 복음으로 세워진

설명:
생명과 소망이 넘치는 교회
```

**팁:**
- 메인 제목에서 `\n`을 입력하면 줄바꿈이 적용됩니다
- 미리보기에서 실제 모습을 확인하세요

**반영 확인:**
- 메인 페이지 맨 위의 큰 배너

## 💡 사용 팁

### ✅ 저장 후 확인하기
1. 수정 후 **"저장"** 버튼 클릭
2. 초록색 메시지 확인: "✓ 저장되었습니다!"
3. 새 탭에서 홈페이지 열기 (상단의 "🌐 사이트 보기" 클릭)
4. 브라우저 **새로고침 (F5)** 하여 변경사항 확인

### ✅ 여러 항목 수정하기
- 각 탭마다 별도로 저장해야 합니다
- 예: 교회 정보 수정 → 저장 → 담임목사 정보 수정 → 저장

### ✅ 실수로 잘못 입력했다면
- 브라우저를 새로고침하면 저장 전 상태로 돌아갑니다
- 저장 후에는 다시 수정해서 저장하세요

### ✅ 모바일에서도 수정 가능
- 관리자 페이지는 모바일에서도 접속 가능합니다
- 다만 화면이 작아 PC 사용을 권장합니다

## 🚀 빠른 시작 가이드

### 처음 사용하시는 경우
1. 관리자 페이지 접속 (비밀번호: `joosung2025`)
2. **교회 기본 정보** 탭에서 전화번호, 이메일, 주소 확인/수정
3. **담임목사** 탭에서 목사님 정보 확인/수정
4. **예배 시간** 탭에서 예배 시간 확인/수정
5. 각 탭마다 **"저장"** 버튼 클릭
6. 홈페이지 새로고침하여 확인

### 팝업 배너 만들기
1. **팝업 배너** 탭 클릭
2. ✅ "팝업 활성화" 체크
3. 제목, 내용, 버튼 정보 입력
4. **"저장"** 클릭
5. 미리보기에서 확인
6. 홈페이지 접속하여 팝업 확인

### 메인 배너 수정하기
1. **메인 배너** 탭 클릭
2. 제목, 부제목, 설명 수정
3. 미리보기에서 확인
4. **"저장"** 클릭
5. 홈페이지 접속하여 배너 확인

## ❓ 자주 묻는 질문

### Q: 저장했는데 변경사항이 안 보여요
**A**: 브라우저를 **새로고침 (F5 또는 Ctrl+R)** 해보세요. 캐시 때문에 이전 내용이 보일 수 있습니다.

### Q: 팝업을 끄고 싶어요
**A**: 관리자 페이지 > 팝업 배너 탭 > "팝업 활성화" 체크 해제 > 저장

### Q: 예배 시간이 변경되었어요
**A**: 관리자 페이지 > 예배 시간 탭 > 해당 예배 시간 수정 > 저장

### Q: 담임목사님이 바뀌었어요
**A**: 관리자 페이지 > 담임목사 탭 > 성함, 학력, 메시지 수정 > 저장

### Q: 전화번호나 이메일이 바뀌었어요
**A**: 관리자 페이지 > 교회 기본 정보 탭 > 해당 정보 수정 > 저장
